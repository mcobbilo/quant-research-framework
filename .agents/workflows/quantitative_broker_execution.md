1. **Verify Egress Sandbox**: Confirm the local `run_zeroclaw_prm.sh` is actively enforcing the `sandbox-exec` egress firewall to prevent exfiltration.
2. **Key Scope Check**: Ensure `os.getenv("ALPACA_KEY_ROLE")` is structurally locked to `RESTRICTED_TRADE_ONLY`. Reject the payload otherwise.
3. **Kelly Sizing Audit**: Mathematically confirm the position size generated by `calc_kelly()` specifically honors the 2.0x Margin Cap absolute limit.
4. **Execution Dry-Run**: Execute `openalice.py` exclusively with `staged` (dry-run) vectors prior to executing a physical API `commit()`.
5. **Log Artifact**: Write the final trade confirmation cryptographic hashes into the localized memory database (`local_persistence.py`).
